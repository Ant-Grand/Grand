 package org.apache.commons.logging;

 /**
 66   
  * <p>A simple logging interface abstracting logging APIs.  In order to be
 67   
  * instantiated successfully by <code>LogFactory</code>, classes that implement
 68   
  * this interface must have a constructor that takes a single String
 69   
  * parameter representing the "name" of this Log.</p>
 70   
  *
 71   
  * <p>The six logging levels used by <code>Log</code> are (in order):</p>
 72   
  * <ol>
 73   
  * <li>trace (the least serious)</li>
 74   
  * <li>debug</li>
 75   
  * <li>info</li>
 76   
  * <li>warn</li>
 77   
  * <li>error</li>
 78   
  * <li>fatal (the most serious)</li>
 79   
  * </ol>
 80   
  * <p>The mapping of these log levels to the concepts used by the underlying
 81   
  * logging system is implementation dependent.
 82   
  * The implementation should ensure, though, that this ordering behaves
 83   
  * as expected.</p>
 84   
  *
 85   
  * <p>Performance is often a logging concern.
 86   
  * By examining the appropriate property,
 87   
  * a component can avoid expensive operations (producing information
 88   
  * to be logged).</p>
 89   
  *
 90   
  * <p>For example,</p>
 91   
  * <pre>
 92   
  *    if (log.isDebugEnabled()) {
 93   
  *        ... do something expensive ...
 94   
  *        log.debug(theResult);
 95   
  *    }
 96   
  * </pre>
 97   
  *
 98   
  * <p>Configuration of the underlying logging system will generally be done
 99   
  * external to the Logging APIs, through whatever mechanism is supported by
 100   
  * that system.</p>
 101   
  *
 102   
  * <p>Note: this package has been copied from
 103   
  * <a href="http://jakarta.apache.org/commons/logging/">Apache Commons Logging</a>
 104   
  * in order to introduce dependencies between Grand's core and 3rd party
 105   
  * libraries.</p>
 106   
  *
 107   
  * @author <a href="mailto:sanders@apache.org">Scott Sanders</a>
 108   
  * @author Rod Waldhoff
 109   
  * @version $Id: Log.java,v 1.14 2003/03/30 23:42:36 craigmcc Exp $
 110   
  */
