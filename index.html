About Grand: Graphical Representation of ANt Dependencies

Grand is a tool to create a visual representation of Ant target dependencies. It works by taking an Ant build file and creating a "dot" file. It differs from the existing tools by relying on the Ant API rather than XML parsing to get the dependencies.
